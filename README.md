## 本地预览

在仓库根目录运行一个简单的静态服务器进行预览：

```bash
# 使用 Python 3
python3 -m http.server 8000

# 然后在浏览器打开：http://localhost:8000
```

文件说明：
- `index.html`：网站首页。
- `css/styles.css`：样式表。
- `js/script.js`：小的交互脚本（联系人表单提示）。
